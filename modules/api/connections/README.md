# UdaConnect Conections Service

UdaConnect Connections service (REST, gRPC & Kafka)

## Installation

Local:

```
$ pip install -r requirements.txt
```

Docker:

```
$ docker build -t nd064-message-passing:connections .
```

## Run Local

```
$ python wsgi.py
```

## OpenAPI Documentation

* http://127.0.0.1:5000/
